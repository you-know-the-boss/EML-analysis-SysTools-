# EML Forensics & Structure Analyzer

This is a single-page HTML application designed for analyzing EML files. It provides a dashboard, an interactive anatomy viewer, an AI assistant, and research summaries.

## Features

- **Forensic Metrics**: Quantitative analysis of EML file composition and common threat vectors.
- **Interactive EML Structure**: Hover over code blocks to decode the function of each segment in a raw EML file.
- **AI Forensic Assistant**: Powered by Gemini LLM. Analyze headers for threats or generate scenario simulations.
- **Research & Video Summaries**: Key takeaways from research materials.
- **Header Reference**: Searchable list of technical terms.

## Usage

Simply open `index.html` in a web browser.

## Deployment

This project is ready to be deployed on Vercel. A `vercel.json` configuration file is included.
